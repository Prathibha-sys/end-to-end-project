# end-to-end-project
House Price Prediction – End-to-End Data Analytics & ML Project
Predicting house prices using Machine Learning with SQL, Python, and Power BI.
Project Overview:
This project builds an end-to-end House Price Prediction system using:
SQL for data storage & querying
Python for data cleaning & EDA
Power BI for interactive dashboard
Excel for Business Analysis & What-If Analysis
Machine Learning for Price Prediction
